# Dépôt "Auto, boulot, marmots, dodo"

Ce dépôt contient les scripts utilisés dans la thèse de M. Guinepain, _Auto, boulot, marmots, dodo ? Une géographie sociale de l'organisation spatio-temporelle des journées de travail et des mobilités du quotidien._, dont la date de soutenance sera annoncée prochainement.

La liste des scripts principaux est dans le répertoire racine. Les scripts d'analyse sont contnus dans le répertoire "Scripts", mais doivent être exécutés depuis le dossier racine (sélectionné avec `getwd()` dans R).
